# `@hokulea/config-typescript`

Typescript configuration for your packagess

## Installation

1) Add this config:

```json
{
  "devDependencies": {
    "@hokulea/config-typescript": "*"
  }
}
```

2) Use in your `tsconfig.json`:

```json
{
  "extends": "@hokulea/config-typescript/tsconfig.json"
}
```
